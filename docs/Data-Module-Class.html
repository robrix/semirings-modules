    class (Semiring r, Semigroup m) => Module r m where Source #

    A left r-module over a Semiring r.

    Laws:

    Associativity of <> (the Semigroup law):

      a <> (b <> c) = (a <> b) <> c

    Identity of <> (the Monoid law, if m is a Monoid):

      zero <> a    = a
  a    <> zero = a

    Left-distributivity of ><< over m <>:

      r ><< (x <> y) = r ><< x <> r ><< y

    Left-distributivity of r <> over ><<:

      (r <> s) ><< x = r ><< x <> s ><< x

    Left-distributivity of >< over ><<:

      (r >< s) ><< x = r ><< (s ><< x)

    Left-identity of ><<, if r is Unital:

      one ><< a = a

    (Applicative f, Semiring r) => Module r (App f r) Source #

    Note that App’s adherence to the distributivity laws depends on the behaviour of f. For example, 'App ZipList' is lawful, while 'App []' is not.

    Left-distributivity of ><< over <>:

    r ><< (x <> y) == r ><< x <> (r :: Boolean) ><< (y :: App ZipList Boolean)

    Left-distributivity of <> over ><<:

    (r <> s) ><< x == r ><< x <> (s :: Boolean) ><< (x :: App ZipList Boolean)

    Left-distributivity of >< over ><<:

    (r >< s) ><< x == r ><< ((s :: Boolean) ><< (x :: App ZipList Boolean))

    Left-identity of ><<:

    (one :: Boolean) ><< a == (a :: App ZipList Boolean)
